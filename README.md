  Built with a focus on aesthetics, performance, and privacy.
</p>

## ‚ú® Features

- **Local-First & Secure**: Everything runs locally on your machine. No data ever leaves your computer.
- **Interactive Visualization**: Explore your database schema with an intuitive graph interface.
- **Smart Layouts**: Automatically organizes your tables with hierarchical or forced-directed layouts.
- **Project Dashboard**: Manage multiple database connections and local SQL folders.
- **Search & Filter**: Quickly find tables, views, and relationships with advanced filtering.
- **Export Options**: Export your schema diagrams as high-quality PNG or SVG images.
- **Dark Mode**: Beautiful execution of Supabase-inspired dark theme.

## üöÄ Getting Started

### Installation

1. Download the latest installer from the [Releases](https://github.com/jenithpaul/pg-studio/releases) page.
2. Run the installer (`PG Studio Setup 1.0.0.exe`).
3. Launch the application from your Desktop or Start Menu.

### Development

To run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/jenithpaul/pg-studio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server (runs both UI and Desktop):
   ```bash
   npm run dev
   ```
